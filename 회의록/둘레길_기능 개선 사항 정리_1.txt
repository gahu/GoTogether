  ○ 관리방안
    - 오프소스 프로젝트 경진대회 이후 지속적인 활동 및 앱 기능 개선이 이루어질 수 있는 방안을 자유롭게 서술

기능 개선 사항
	1. 네비게이션이 부적절함
		- 메뉴가 어떤 컨텐츠를 제공하는지 들어가기 전에는 알 수 없다
		- 주요 지점 상세보기에서 사진을 클릭했을 때 서울둘레길 홈페이지로 연결되는데 의도파악이 불분명

	2. 주 사용자를 고려하지 않은 UI/UX
		- 폰트 크기가 작고 줄간격이 좁다. 가독성이 좋지 않다

	3. 사용성 증대
		# 문의전화 바로가기
		- 안내지도, 스탬프북 물량이 없을 수 있으니 문의전화 후 방문하라는 유의사항이 써 있음
		- 전화번호를 눌렀을 때 바로 안내센터로 전화걸 수 있다면 사용성이 증대될 것으로 예상
		# 찾아오시는길 바로가기
		- 찾아오시는 길 클릭시 네이버 길찾기로 이동
		# 주요지점 사진 수 증가
		- 해당 지점에 대한 사진도 하나밖에 안나오는데 여러개를 슬라이드 형태로 볼 수 있으면 좋을 것 같다
		# 주요지점 리스트 구성 변경
		- 주요 지점을 화장실별, 볼거리별, 카페별로 필터링해서 보기 쉽게 재구성한다
		# 주요지점에 카페, 맛집 리스트 추가
		- 주변 카페, 맛집 추천으로 지역 상권 활성화
		# 둘레길 영상 메뉴 개편
		- 둘레길 소개와는 무관한 영상이 있다. 메뉴에 실속이 없다
		- 코스 상세보기에 유튜브와 네이버 블로그에서 해당 코스를 검색해 참고할만한 자료만 나오게 한다
		# QR코드 리더기 앱에 내장
		- QR코드 안내판도 볼 수 있는데 리더기가 내장되어있지 않아 사용성이 떨어진다.
		# 카메라 기능 내장
		- 카메라로 들어갈 필요없이 앱 자체에서 카메라기능을 만들어서 앱이용하다가 사진찍을수 있게 함

	4. 너무 긴 실행시간
		- 주요 지점에서 상세보기 클릭시 UI 생성 시간이 너무 길다

	5. 스탬프 관리 기능 개선
		- 현재 앱 삭제시 스탬프 기록 소멸
		- 원하는 사용자에 한해 회원제로 운영하여 스탬프에 대해서 영구적 관리
		- 스탬프 관련 자료 pdf로 한번에 묶어서 제공

	6. 회원제를 통한 개인에 최적화된 기능 제공
		- 개인에 최적화된 코스 추천 기능

	7. 현재 커뮤니티에서 제공하는 기능을 앱으로 이전
		- 주요 기능 : 아카데미(자원봉사자 관리), 정모, 정기적 원정대를 모집하고 관리함
		- 카페에서 앱으로 이러한 기능들을 이전할시 얻을 수 있는 행정적 기대효과는?
		- 모니터링 보고 앱을 통해서 쉽고 간편하게!
		- 앱으로 정모, 원정대 모집 등 푸쉬 알림 제공으로 둘레길 참여 유도

	8. 안전 관련 기능 추가
		- 위치 추적 기능 등을 활용해 실종시 위치 추적에 도움을 줌
		- 실제 답사를 통하여 낙석주의구간 및 안전 지대에 취약한 부분을 확인하여 
		  개발에 적용 시킬 수 있도록 계획
		- (양천구 네비게이션 정보 이용하여) 코스 중 위험지역일 경우 푸쉬 알림 제공
		* 구간별 위험지역, 보행자 취약지역, 편익, 안전시설 등 둘레길 앱보다 다양한 지점정보와 안전기능 제공
		  하지만 설치과정이 굉장히 복잡하여 어르신들이 사용하기에는 무리가 있을 듯으로 사료됨

	

활동계획 : 프로젝트를 진행하는데 있어서 어떤 활동을
할꺼고 어떤 계획으로 진행할꺼야?

	- (개선해야 할 기능은 이미 1차 확인했다는 전제)
	- 답사를 통해 둘레길에서의 실제 사용성 확인
		-> 둘레길 산행시 사용성, 안전 관련하여 취약한 부분 확인
	- 앱사용 실태 조사(둘레길 이용자, 100인 원정대 등)
		-> 실제 사용자가 원하는 요구 사항 확인
	- 1차 개발 착수(UI 개선, 기존 기능 개선, 기능 추가 순)
	- 베타 테스터 모집 및 답사를 통한 기능 검증
		-> 테스트 케이스 명세서 제작하여 관리
	- 베타 테스트를 통한 피드백을 기반으로 한 2차 개발
	- 둘레길 앱 홍보 및 마케팅
		-> 둘레길 입구나 하산지역 주변에 어플리케이션 홍보용 푯말 설치를 통하여 둘레길 어플에 대한 홍보
		-> 둘레길 카페와 홈페이지에 홍보배너 제작
	- 최종 수정 및 제출
		=> 간트차트 형태로 제작

		-> 시민 참여단 전용 요구분석명세서 제작하여 지속적으로 관리
